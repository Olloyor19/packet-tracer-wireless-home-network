## Description
This project shows a basic home network configuration using Cisco Packet Tracer.  
The network includes wired and wireless devices connected through a home wireless router.

## What was done
- Connected cable modem and TV using coaxial cables
- Connected two PCs to the router using Ethernet cables
- Configured the wireless router using GUI
- Enabled DHCP and limited users to 10
- Changed default admin password
- Created a secure Wi-Fi network (SSID: MyHome, WPA2)
- Connected a laptop to the wireless network
- Tested internet connectivity on all devices

## Tools Used
- Cisco Packet Tracer
- Basic networking (DHCP, LAN, WLAN)

## Result
All devices successfully received IP addresses and accessed the internet.
